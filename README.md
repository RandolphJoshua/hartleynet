# HartleyNet — Streamlit Demo

A simple Streamlit web app that wraps the **HartleyNet** model from our undergraduate
thesis. Upload an image and the app predicts whether the image is AI-generated or real.

> Status: work in progress. UI scaffolding and model integration coming next.

## Stack
- Python 3.10+
- PyTorch
- Streamlit

## Run locally (once code is in)
```bash
pip install -r requirements.txt
streamlit run app.py
```
